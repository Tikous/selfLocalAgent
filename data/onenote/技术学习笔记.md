## React开发

### Hooks使用
- **useState**: 管理组件状态
- **useEffect**: 处理副作用
- **useContext**: 使用Context API
- **useReducer**: 复杂状态管理
- **useMemo**: 性能优化
- **useCallback**: 函数缓存

### 性能优化
1. 使用React.memo包装组件
2. 合理使用useMemo和useCallback
3. 避免不必要的重新渲染
4. 代码分割和懒加载
